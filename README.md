### Usage

> Single Page

`config/index.js` -> singlePage: `true`

```
src/目录的必要结构
    - index.html
    - index.js
```

> Multi Page

`config/index.js` -> singlePage: `false`

```
src/目录的必要结构
    - Folder1
        - index.html
        - index.js
    - Folder2
        - index.html
        - index.js
```
